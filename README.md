# JOSHELE - Living & More

Modern website for JOSHELE Klosters - Decoration & Women's Fashion.

## Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- Formspree (Contact Form)
- Vercel Deployment

## Features

- Responsive Design
- Google Maps Integration
- Contact Form (Formspree)
- Instagram Integration
- Fast & Optimized

## Deployment

Deployed on Vercel: https://joshele.ch
